API

Api URL: /api/books


Avaliable queries

?title="" Filters books that CONTAINS provided title

?author="" Filters books that has provided Author

?publication_language="" Filters books that has provided publication language

?from_date="" Filters book that has published date greater or equal than provided from_date

?to_date="" Filters book that has published date, less or equal than provided to_date

Example: /api/books?title="flowers"?from_date=1990?to_date=2004


POST request /api/books to create new record
